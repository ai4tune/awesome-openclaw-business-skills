# Getting Started

## What This Repository Is

This is not a generic prompt dump.

It is a curated sample repository of business-oriented OpenClaw skills designed for practical AI adoption inside real teams and business workflows.

## How To Use It

1. Start from the catalog in the root README.
2. Pick a skill closest to your workflow.
3. Read the skill README before opening `skill.md`.
4. Check example input and output to understand the expected format.
5. Adapt the baseline version to your business context.

## What To Expect

Each public skill aims to be:

- Easy to scan
- Useful without a large setup
- Safe to adapt
- Clear about scope and limits

## What This Repository Is Not

- Not a full enterprise product
- Not a confidential client delivery archive
- Not an everything-bucket of prompts
