# Training Summary

## What It Does

Transforms training notes or workshop transcripts into a reusable summary, FAQ, and action checklist.

## Best For

- Internal enablement
- Workshop recap
- Team knowledge capture

## Input Format

- Training transcript
- Speaker notes
- Workshop outline

## Output Format

- Key concepts
- FAQ
- Recommended actions
- Follow-up resources

## Example Input

See [examples/input.md](examples/input.md).

## Example Output

See [examples/output.md](examples/output.md).

## Limitations

- Technical accuracy depends on source quality
- FAQ items may need review for policy-sensitive topics
- Not a replacement for a full training manual

- Quiz generation
- Learning path bundling
