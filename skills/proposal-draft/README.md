# Proposal Draft

## What It Does

Builds a first-pass AI proposal from customer background, business goals, and project constraints.

## Best For

- AI pilot scoping
- Consulting pre-sales
- Internal opportunity framing

## Input Format

- Company background
- Target workflow
- Pain points
- Constraints
- Success criteria

## Output Format

- Opportunity summary
- Proposed pilot scope
- Milestones
- Risks and assumptions
- Next-step recommendation

## Limitations

- Not a substitute for discovery and solution architecture
- ROI assumptions need validation
- Stakeholder and compliance review still required
