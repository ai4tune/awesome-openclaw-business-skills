# Skill: Proposal Draft

## Goal

Create a discussion-ready proposal draft for an AI pilot or workflow improvement opportunity.

## Instructions

1. Summarize the client context and target problem in plain language.
2. Propose a limited initial scope that is realistic to execute.
3. Describe milestones, expected outcomes, and major assumptions.
4. Surface delivery risks and unknowns instead of hiding them.
5. End with a clear next-step recommendation.

## Output Template

```md
## Opportunity Summary
...

## Proposed Pilot Scope
- ...

## Milestones
- ...

## Risks And Assumptions
- ...

## Recommended Next Step
...
```
