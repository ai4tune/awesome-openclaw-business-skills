# Meeting Minutes

## What It Does

Turns a meeting transcript or raw notes into a structured summary with decisions, action items, owners, deadlines, and risks.

## Best For

- Internal syncs
- Project reviews
- Cross-functional coordination meetings

## Input Format

- Transcript
- Bullet notes
- Chat excerpts from the meeting

## Output Format

- Meeting objective
- Key decisions
- Action items with owners
- Open questions
- Risks or blockers

## Limitations

- Weak or incomplete notes reduce output quality
- Owner names and deadlines may need manual confirmation
- Does not replace official project tracking tools
