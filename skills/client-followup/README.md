# Client Follow-up

## What It Does

Turns sales or consulting conversation notes into a structured follow-up view of client needs, concerns, next steps, and suggested response angles.

## Best For

- Sales meetings
- Discovery calls
- Consulting check-ins

## Input Format

- Visit notes
- Chat history
- CRM comments
- Call transcript

## Output Format

- Client goals
- Key concerns
- Buying signals
- Follow-up actions
- Suggested response language

## Limitations

- Should not invent budget or timeline details
- Suggested messaging still needs human review
- Not a replacement for CRM hygiene
