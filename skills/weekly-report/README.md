# Weekly Report

## What It Does

Converts fragmented task notes into a concise weekly report that a manager or stakeholder can skim quickly.

## Best For

- Individual contributor weekly updates
- Team lead summary drafting
- Project status rollups

## Input Format

- Task lists
- Calendar notes
- Meeting bullets
- Chat fragments

## Output Format

- This week completed
- Business impact
- Risks or blockers
- Next week plan

## Example Output

## Limitations

- Requires manual validation for metrics
- Does not infer impact beyond provided evidence
- Sensitive or political updates should be reviewed by the author
