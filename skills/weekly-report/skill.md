# Skill: Weekly Report

## Goal

Turn raw weekly work logs into a readable management update.

## Instructions

1. Group related work into a few meaningful themes.
2. Distinguish completed work from in-progress items.
3. Summarize visible impact without exaggeration.
4. Call out blockers and dependency risks clearly.
5. End with a realistic plan for the next week.

## Output Template

```md
## This Week
- ...

## Business Impact
- ...

## Risks And Blockers
- ...

## Next Week
- ...
```
