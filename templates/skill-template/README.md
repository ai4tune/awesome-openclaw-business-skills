# Skill Template

Use this template when adding a new business workflow skill to the repository.

## Required Files

```text
skills/<skill-name>/
├── README.md
├── skill.md
└── examples/
    ├── input.md
    └── output.md
```

## Required README Sections

- What it does
- Best for
- Input format
- Output format
- Example input
- Example output
- Limitations
- Upgrade ideas
- Related use cases
